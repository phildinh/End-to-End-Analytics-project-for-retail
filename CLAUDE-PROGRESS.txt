      - generate_batch_id() — returns YYYYMMDD_HHmmss string

[ ] loader/load_full.py
      - Reads all CSVs from /data/
      - Adds loaded_at = datetime.now() (once per batch)
      - Adds batch_id = generate_batch_id() (once per batch)

      - Logs row counts per table
      - Triggers dbt run on completion

      - Appends only — no truncate
      - dbt handles deduplication via surrogate key
      - Logs new row counts per table
